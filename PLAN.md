## 通常の検証

Wikiやスクリプトを更新したら、少なくとも以下を実行する。

```bash
python3 scripts/lint-wiki.py
git diff --check
python3 -m py_compile scripts/search-archive.py
pnpm build && pnpm check:pages-links
```

`PLAN.md` だけの更新なら `pnpm build` は必須ではないが、Wiki本文やリンクを触った場合は必ず実行する。

## 方針

- 初めての人が読んで理解できることを最優先する
- 日本語で書く
- `raw/` は変更しない
- Slackの私的発言や雑談はWikiに持ち込まない
- 議論の経緯と決定事項を分ける
- 新しいソースを使ったら、根拠が辿れるように `sources:` と関連ページを更新する
- 活動には波があるため、ページ数の網羅よりも、初見者が後から参照する価値と根拠の強さを優先する
